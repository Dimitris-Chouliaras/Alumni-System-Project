## 📝 Περιγραφή
Η εφαρμογή επιδεικνύει τη χρήση της τεχνικής **AJAX (Asynchronous JavaScript and XML)** για την ασύγχρονη φόρτωση και απεικόνιση δεδομένων από δύο διαφορετικές πηγές:
1.  **XML Αρχείο (`alumni.xml`)**: Ανάγνωση δεδομένων με χρήση `DOMParser` και πλοήγηση σε δέντρο XML.
2.  **JSON Αρχείο (`alumni.json`)**: Ανάγνωση δεδομένων με τη χρήση της σύγχρονης εντολής `fetch` και άμεση μετατροπή σε JavaScript Objects.

## 🚀 Χαρακτηριστικά
* **Πολλαπλότητα & Βάθος**: Ο κώδικας πλοηγείται σε σύνθετες δομές δεδομένων (π.χ. `contact -> social_networks -> social`).
* **Dynamic UI**: Χρήση **Template Literals** της JavaScript για τη δυναμική δημιουργία HTML καρτών για κάθε απόφοιτο.
* **Interactive Elements**: 
    * Clickable links για GitHub και LinkedIn (με άνοιγμα σε νέα καρτέλα).
    * Λειτουργία καθαρισμού οθόνης.
    * **Easter Egg**: Ένα "Surprise Button" που επιδεικνύει DOM manipulation και χρήση χρονικών καθυστερήσεων (`setTimeout`).

## 🛠️ Τεχνολογίες
* **HTML5 / CSS3** (Flexbox layout)
* **JavaScript (ES6+)**
* **AJAX / Fetch API**
* **Font Awesome** (για τα εικονίδια στο footer)

# 🎓 Alumni Management System (AJAX Project)

This web application was developed as part of the **MSc in Software Engineering for Internet and Mobile Applications** at the University of Thessaly.

## 🔗 Live Demo
Check out the live application here: 
[**GitHub Pages Link**](https://dimitris-chouliaras.github.io/Alumni-System-Project/)

## 📝 Description
The application demonstrates the use of **AJAX (Asynchronous JavaScript and XML)** techniques to fetch and display data asynchronously from two different data sources:
1.  **XML File (`alumni.xml`)**: Data parsing using `DOMParser` and XML tree navigation.
2.  **JSON File (`alumni.json`)**: Modern data fetching using the `fetch` API and direct mapping to JavaScript Objects.

## 🚀 Key Features
* **Nesting & Complexity**: The code navigates through complex data structures (e.g., `contact -> social_networks -> social`).
* **Dynamic UI**: Uses JavaScript **Template Literals** to dynamically generate HTML cards for each alumnus.
* **Interactive Elements**: 
    * Clickable links for GitHub and LinkedIn (opening in a new tab).
    * Screen clearing functionality.
    * **Easter Egg**: A "Surprise Button" demonstrating DOM manipulation and the use of time delays (`setTimeout`).

## 🛠️ Technologies Used
* **HTML5 / CSS3** (Flexbox layout)
* **JavaScript (ES6+)**
* **AJAX / Fetch API**
* **Font Awesome** (for social icons in the footer)
